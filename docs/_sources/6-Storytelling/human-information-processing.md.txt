# 人的信息处理模型

信息处理模型是一种以计算机信息处理过程为类比，解释人类如何接收、存储、处理和输出信息的认知模型。该理论认为人类认知系统可以被看作一个处理信息的系统，通过输入、编码、存储、检索等步骤完成学习与记忆。



## 主要组成部分

信息处理模型通常包含三个主要阶段：

### 感官记忆（Sensory Memory）

- 信息来源于外界刺激（视觉、听觉等）
- 保持时间极短（0.5–3 秒）
- 信息未注意即快速遗忘

示例：看到路上闪过的汽车，但未加注意便忘记。

### 短时记忆 / 工作记忆（Short-term / Working Memory）
​	•	信息经过注意进入此阶段
​	•	容量有限（约 7±2 个单位）
​	•	持续时间短（10–30 秒）
​	•	通过复述可转入长期记忆

示例：临时记住一个电话号；听老师讲课时记录重点。



### 长期记忆（Long-term Memory）

- 信息被编码并长期存储
- 容量巨大，无明显限制
- 可分为情景记忆（事件）、语义记忆（知识）、程序性记忆（技能）

示例：会骑自行车、记得中国的首都、记得小时候的生日。



### 信息加工的重要机制

| **机制** | **说明**                     | **示例**               |
| -------- | ---------------------------- | ---------------------- |
| 注意     | 决定感觉信息是否进入短时记忆 | 专注听课 vs 被手机分心 |
| 编码     | 将信息转化为可以存储的形式   | 通过联想法记单词       |
| 复述     | 保持/强化短时记忆            | 不断读电话号码         |
| 提取     | 从长期记忆中恢复信息         | 考试时回忆知识点       |



### 与计算机类比

| **认知阶段** | **计算机类比**          | **人的例子**     |
| ------------ | ----------------------- | ---------------- |
| 感觉记忆     | 输入设备（键盘/摄像头） | 眼睛看到图像     |
| 短时记忆     | 内存RAM                 | 正在做算术题     |
| 长期记忆     | 硬盘/云盘               | 记住英语语法规则 |
