# 描述性统计常见指标

## 集中趋势
- 平均值 (mean)：全体数据的大致水平  
- 中位数 (median)：一半人高于它，一半人低于它  
- 众数 (mode)：出现次数最多的值  
这些指标告诉我们“数据的中心在哪里”  

## 离散程度
- 方差 / 标准差：数据分布得紧还是松  
- 极差：最高分和最低分的差距  
- 四分位差：中间 50% 数据的范围  
这些帮助我们理解“差异大不大”  

## 分布形态
- 偏态（左偏/右偏）：数据分布的倾斜方向  
- 峰态（扁平/尖锐）：分布的集中程度  
例如：班级成绩左偏，说明高分多；右偏，说明低分多  

## 频率与比例
- 各区间人数、比例（直方图/饼图/柱状图）  
- 例如：多少人及格，多少人优秀  
这样能直观展示“各部分占比”  

## 相关性（简单描述）
- 两个变量之间的线性关系，如“学习时间和成绩的相关性”  
- 这里只是描述性，不是因果推断
