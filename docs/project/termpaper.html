期末论文¶

综述¶

可就本学期感兴趣的知识点进一步深入探索，然后撰写一份综述报告，例如：

  1. 图文阅读速度的差异

  2. 读者的注意力分配机制

  3. 读者的前注意力分配机制

  4. 不同配色对于读者观感的影响

  5. 人类对于差异感知的精度研究（色彩深浅表示的差异，回归线斜率表示的差异，条形图长短表示的差异等等）

可自选感兴趣的内容，但需与数据可视化有关。

可以使用ChatGPT DeepResearch, Google Gemini DeepResearch , Grok DeepSearch，或者SurveyX等工具作为辅助，但不可由AI自动生成，需有自己的见解。

图表¶

介绍一个复杂的图表，例如：Voronoi diagram（沃罗诺伊图）、Streamgraph （流图）、 Parallel Coordinates Plot （平行坐标图）等等。

内容结构：

  1. 图表形状

  2. 功能与用途

  3. 能使用该图的场景和数据

  4. 使用任意工具绘制出该图（需要展示完成过程）

教程¶

A: 课程工具的深入¶

可就课程上介绍的工具，进一步深挖，并撰写一份使用教程，例如：

  1. 使用提示词生成 Seaborn 可视化代码

  2. Pandas高级数据处理功能

B: 选择一个新工具¶

撰写一份本课程未涉及，但是与数据可视化有关的教程：

  1. 使用Vega通过文字描述可视化数据

  2. 使用Apache Echarts 进行数据可视化

  3. 使用 D3.js 绘制动态可视化网页

  4. 使用Claude Code辅助数据可视化

格式要求¶

  • 字数：3000字左右（6 页 A4纸篇幅）

  • 推荐使用Markdown撰写，LaTex或Word亦可

  • 需符合论文引用格式
